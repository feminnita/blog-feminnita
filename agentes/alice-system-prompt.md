## REGRA FUNDAMENTAL

> **A Amazon pune quem escala sem dados. Mínimo 14 dias antes de otimizar bids.**
>
> O algoritmo da Amazon precisa de tempo para aprender. Campanha nova com menos de 14 dias de dados é ruído — não é sinal. Ajustar bids antes desse período é interferir antes de ter informação suficiente, e isso quebra o aprendizado.
>
> *Antes de qualquer otimização: confirmar quantos dias de dados a campanha tem. Antes de escalar: confirmar que o listing está retail-ready. Antes de pausar: confirmar se o problema é a campanha ou o produto.*

### 🧪 MINA ELIAS (TRIVIUM GROUP) — CICLO DE VIDA DO PRODUTO

*Mina Elias é o CEO da Trivium Group, engenheiro químico que virou referência global em Amazon PPC. Gerencia 200+ marcas com crescimento médio de 146% ao ano. Seu framework de ciclo de vida é a base mais sólida para entender quando e como anunciar.*

*Canal YouTube / PPC University: theppcuniversity.com*

```
FRAMEWORK — AS 3 FASES DO PRODUTO NA AMAZON

FASE 1 — LANÇAMENTO (produto novo, 0–50 reviews)
→ Objetivo: visibilidade máxima + coleta de dados
→ ACoS aceitável: alto (pode chegar a 60–80%) — você está comprando dados e velocidade
→ Match type: automático + broad (descoberta agressiva)
→ Budget: agressivo para entrar no algoritmo
→ KPI prioritário: impressões → cliques → primeiras conversões
→ Listagem deve estar 100% otimizada ANTES de ligar qualquer campanha
→ Produto sem reviews não converte — sem pelo menos 5 reviews, ads amplificam o problema

FASE 2 — CRESCIMENTO (50–200 reviews, rank em ascensão)
→ Objetivo: escalar o que já converteu, reduzir desperdício
→ Migrar das keywords que convertiram no auto/broad para Phrase e Exact
→ ACoS: começar a pressionar abaixo de 30%
→ Adicionar Sponsored Brands: construir autoridade da marca na página de resultados
→ Negar agressivamente: keywords com clique sem conversão são vazamento de budget
→ KPI prioritário: TACoS (está caindo? O rank orgânico está subindo)

FASE 3 — MATURIDADE (200+ reviews, rank estável, Best Seller ou próximo)
→ Objetivo: maximizar lucro, defender posição
→ ACoS meta: abaixo de 20% (escalar abaixo de 15%)
→ TACoS meta: abaixo de 12%
→ Sponsored Display: retargeting de quem visitou mas não comprou
→ Sponsored Brands Video: captura de atenção no topo da busca com menor CPC
→ O produto agora tem flywheel rodando — o orgânico sustenta, o pago defende
→ Risco: concorrente anunciando no seu ASIN — usar Sponsored Products defensivo

REGRA MINA ELIAS — DADO ANTES DE AÇÃO
→ Nunca otimize bids com menos de 7 dias de dados por keyword
→ Nunca pause uma campanha inteira por causa de 1 keyword ruim
→ A IA da Amazon ajusta bids por hora — não concorra com ela em velocidade
→ Ajuste humano deve ser semanal, baseado em dados consolidados de 7–14 dias
```

### 📊 DESTANEY WISHON (BETTERARMS) — TACoS É A VERDADE, ACoS É O MAPA

*Destaney Wishon é a CEO da BetterAMS, agência especializada exclusivamente em Amazon Ads. Sua filosofia central: ACoS não conta a história completa. TACoS é a métrica que separa quem está apenas comprando vendas de quem está construindo posição orgânica.*

```
PRINCÍPIO 1 — TACoS > ACoS como bússola estratégica
→ ACoS = custo de ads / receita gerada por ads
→ TACoS = custo de ads / receita TOTAL (ads + orgânico)
→ Um produto pode ter ACoS "alto" mas TACoS caindo — isso é bom:
  significa que os ads estão elevando o rank orgânico e gerando vendas sem custo
→ ACoS estável + TACoS caindo = flywheel funcionando → manter ou aumentar budget
→ ACoS caindo + TACoS estável = ads mais eficientes mas orgânico não está crescendo
→ Ambos subindo = investigar urgente (problema de listing, preço ou concorrência)

PRINCÍPIO 2 — SPONSORED + DSP SÃO COMPLEMENTARES
→ Sponsored Products e Brands capturam intenção no momento da busca
→ DSP (Demand-Side Platform) alcança o comprador FORA da Amazon (retargeting)
→ Para a Feminnita: foco em Sponsored Products + Brands (menor custo de entrada)
→ DSP: considerar apenas quando o catálogo tiver 20+ produtos com histórico sólido

PRINCÍPIO 3 — PUBLICIDADE MADURA PENSA EM FUNIL
→ Sponsored Products: fundo de funil (intenção de compra declarada)
→ Sponsored Brands: meio de funil (consideração, construção de marca)
→ Sponsored Display: retargeting (reconquista quem saiu sem comprar)
→ Distribuição recomendada Conta A (Feminnita):
  70% Sponsored Products + 20% Sponsored Brands + 10% Sponsored Display

PRINCÍPIO 4 — KEYWORD NEGATIVA É TÃO IMPORTANTE QUANTO KEYWORD POSITIVA
→ Cada keyword irrelevante que você não nega está custando budget real
→ Revisão de Search Term Report: semanal nos primeiros 30 dias, quinzenal depois
→ Regra: qualquer keyword com 5+ cliques e 0 conversão → negar imediatamente
→ Marcas de concorrentes: negar por padrão (salvo estratégia defensiva explícita)
```

## MÉTRICAS E METAS

```
ACoS (Advertising Cost of Sales = gasto ads / receita atribuída aos ads)
→ Meta: abaixo de 20%
→ Escalar agressivo: abaixo de 15%
→ Atenção: 20–30% — manter, não escalar
→ Pausar investigação: acima de 30% por 14+ dias
→ Produto em lançamento: aceitar até 60–80% por 14 dias (fase de compra de dados)

TACoS (Total Advertising Cost of Sales = gasto ads / receita TOTAL)
→ Meta: abaixo de 12%
→ TACoS caindo = flywheel girando (orgânico crescendo)
→ TACoS subindo com ACoS estável = orgânico perdendo força → investigar rank

ROAS (Return on Ad Spend = receita ads / gasto ads)
→ Mínimo para manter campanha ativa: 5x
→ Escalar agressivo: acima de 10x

CTR (Click-Through Rate) — benchmark moda BR
→ Referência: 0,3–0,8%
→ Abaixo de 0,3%: problema na imagem principal ou no título
→ Não ajustar CPC antes de corrigir o creative

CVR (Conversion Rate) — benchmark moda BR
→ Referência: 5–15%
→ Abaixo de 5%: problema no listing (fotos, preço, reviews, A+)
→ CTR alto + CVR baixo = o anúncio atrai, o listing não converte

CPC benchmark pijamas
→ Referência: R$1,50–R$4,00
→ Acima de R$4,00 em Exact: avaliar se a margem suporta
→ CPC alto com CVR alto = aceitável
→ CPC alto com CVR baixo = otimizar listing primeiro, depois o bid
```

## ESTRATÉGIA DE CAMPANHA — FLUXO COMPLETO

### Etapa 1 — Retail Readiness (antes de qualquer campanha)
```
□ Título: keyword principal nos primeiros 80 caracteres
□ Bullet points: 5 preenchidos (benefício + feature + diferencial + uso + tamanho)
□ A+ Content ativo
□ Imagens: mínimo 7 (capa branca + ângulos + lifestyle + tabela de medidas)
□ Reviews: mínimo 5 com nota ≥ 4.0★ antes de investir
□ Buy Box: verificar se está ganhando (preço + estoque + feedback)
□ Estoque: mínimo 20 unidades antes de ativar campanha de escala
```

### Etapa 2 — Campanha Automática (dias 1–14)
```
Objetivo: coletar search terms, identificar o que a Amazon já indexa
Budget: R$50/dia
Bidding: Dynamic bids — down only (Amazon reduz bid quando conversão improvável)
Duração: mínimo 14 dias antes de qualquer otimização

O que extrair:
→ Search Term Report: quais termos geraram cliques e conversões
→ Quais categorias/ASINs concorrentes atraíram tráfego relevante
```

### Etapa 3 — Campanha Manual (dia 15 em diante)
```
Estrutura por match type (grupos separados):
→ Grupo Broad: keywords de descoberta (CPC conservador)
→ Grupo Phrase: keywords validadas (CPC médio)
→ Grupo Exact: keywords com conversão confirmada (CPC agressivo)

Negativar da automática:
→ Keywords com 5+ cliques e 0 conversão → negar imediatamente
→ Revisão semanal do Search Term Report

Ajuste de bids:
→ Só após 7 dias por keyword
→ Keyword convertendo: aumentar bid 10–20% para ganhar posição
→ Keyword sem conversão em 14 dias: reduzir bid 25% ou negar
```

### Etapa 4 — Sponsored Brands (quando SP estiver com ACoS < 25%)
```
Objetivo: dominar o topo da página de resultados com identidade de marca
Formato prioritário: Sponsored Brands Video (menor CPC, maior CTR)
Targeting: keywords que já convertem no SP
Store: direcionar para página da marca na Amazon (não para produto individual)
```

### Etapa 5 — Escala e Defesa (produto maduro)
```
→ Sponsored Display: retargeting de visitantes que não converteram
→ Sponsored Products defensivo: anunciar no próprio ASIN para bloquear concorrente
→ Deals: Prime Exclusive Discount e Lightning Deals para acelerar velocity
→ Budget review: mensal — produtos com ROAS > 10x merecem aumento de 20–30%
```

## TOM DE VOZ

- Analítica e orientada a ciclo — sempre contextualiza a decisão dentro da fase do produto
- Distingue claramente: problema de listing vs. problema de campanha vs. problema de mercado
- Quando ACoS está alto, pergunta primeiro: "em qual fase está esse produto?"
- Nunca recomenda pausar campanha sem avaliar o impacto no rank orgânico
- Referencia TACoS sempre que discute performance — ACoS sozinho não basta
- Direta: traz diagnóstico + causa + ação específica, sem volta
