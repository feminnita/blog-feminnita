### 🤖 SHOPEE ADS ACADEMY — O ALGORITMO É O MESTRE

*A Shopee Ads Academy (ads.shopee.com.br) é a fonte mais atual e autoritativa sobre como o GMV Max funciona. Em julho de 2025, o sistema de lances manuais foi descontinuado. Entender as regras do GMV Max é a habilidade mais crítica de 2025.*

```
GMV MAX — OS DOIS MODOS

MODO 1 — GMV MAX AUTO (Lance Automático)
→ Objetivo: maximizar GMV dentro de um orçamento
→ Algoritmo opera sem meta de ROAS — foco em volume
→ Quando usar: produto novo (sem histórico de conversão)
  ou quando o objetivo é volume de vendas, não margem
→ Risco: ROAS pode ser baixo se o produto não converter bem
→ Estratégia: usar por 7–14 dias para "esquentar" o produto
  antes de migrar para Meta de ROAS

MODO 2 — GMV MAX META DE ROAS (Lance com Objetivo)
→ Objetivo: atingir uma meta de ROAS definida por você
→ Algoritmo otimiza a entrega para atingir o ROAS alvo
→ Quando usar: produto com histórico de vendas e ROAS estabelecido
→ Proteção de ROAS: ativada automaticamente — se o ROAS cair
  abaixo de 70% do alvo, o sistema reduz a entrega para proteger
→ Impulsão Rápida: quando ativada, o algoritmo aceita ROAS até 30%
  abaixo do alvo temporariamente para acelerar volume e coleta de dados

REGRA DE OURO DO GMV MAX:
→ ROAS alvo muito alto = algoritmo tem dificuldade de gastar o budget
  (poucas impressões, poucas conversões)
→ ROAS alvo muito baixo = gasta rápido mas margem insuficiente
→ Encontrar o ROAS alvo calibrado é a principal habilidade do especialista

COMO CALIBRAR O ROAS ALVO:
→ Observe o ROAS histórico do produto nos últimos 14–30 dias
→ Meta inicial: ROAS atual + 10–15% acima (objetivo alcançável)
→ Ajustar a cada 14 dias conforme performance
→ Nunca mudar ROAS alvo com menos de 7 dias de dados — o algoritmo não
  teve tempo suficiente para aprender
```

### 🧪 MINA ELIAS (TRIVIUM GROUP) — CICLO DE VIDA DO PRODUTO ADAPTADO À SHOPEE

*A filosofia de ciclo de vida de Mina Elias (maior especialista em Amazon PPC, CEO da Trivium Group) se aplica diretamente à Shopee: cada fase do produto exige uma estratégia diferente de ads. Adaptado ao GMV Max.*

```
FASE 1 — LANÇAMENTO (produto novo, < 10 avaliações)
→ Ads objetivo: gerar as primeiras vendas para esquentar o produto
→ Usar GMV Max Auto — sem meta de ROAS (foco em volume, não em margem)
→ Budget: R$30–50/dia (mínimo para o algoritmo ter dados suficientes)
→ Duração: 7–14 dias
→ Métricas a observar: impressões crescendo? Cliques? Primeiras conversões?
→ Sem pelo menos 10 reviews: ads amplificam o problema, não a solução
→ Preparação obrigatória antes de ativar: fotos, descrição, preço alinhado
  com categoria

FASE 2 — CRESCIMENTO (10–50 avaliações, produto com tração)
→ Migrar para GMV Max Meta de ROAS
→ ROAS alvo inicial: usar o ROAS médio dos últimos 14 dias da fase automática
→ Adicionar Search Ads com keywords específicas (produto já tem histórico)
→ Testar Discovery Ads para ampliar alcance além da busca direta
→ Ajustar ROAS alvo a cada 14 dias, aumentando gradualmente

FASE 3 — MATURIDADE (50+ avaliações, posição estável na categoria)
→ Objetivo: defender posição + maximizar margem
→ GMV Max Meta de ROAS com alvo agressivo (acima do ROAS médio histórico)
→ Ativar Proteção de ROAS como segurança
→ Flash Sale Ads + campanhas sazonais para picos programados
→ Shop Ads para fortalecer autoridade da loja inteira (não só produto isolado)

REGRA MINA ELIAS ADAPTADA:
→ Nunca trate produto novo igual a produto maduro
→ Produto novo: aceite ROAS baixo no início — você está comprando histórico
→ Produto maduro: exija ROAS acima do alvo — o algoritmo já aprendeu a converter
```

### 📊 DESTANEY WISHON (BETTERARMS) — ROAS COMO BÚSSOLA, NÃO COMO TETO

*Destaney Wishon (CEO BetterAMS, especialista em Amazon Ads) provou que ROAS não deve ser tratado como teto — é uma bússola de navegação. Mesma lógica se aplica ao GMV Max da Shopee.*

```
PRINCÍPIO 1 — ROAS MUITO ALTO PARALISA O ALGORITMO
→ Meta de ROAS elevada demais = o sistema não encontra compradores que atinjam
  esse retorno → baixas impressões → baixo gasto → produto estagnado
→ Sintoma: budget do dia não está sendo gasto. Causa mais comum: ROAS alvo
  alto demais para o estágio do produto

PRINCÍPIO 2 — ROAS ABAIXO DO MÍNIMO É VAZAMENTO, NÃO INVESTIMENTO
→ ROAS < 2x na Shopee = você está gastando mais em ads do que gerando em receita
→ Identificar a causa antes de pausar: problema no produto ou no targeting?
→ Produto com CTR alto e ROAS baixo = problema de conversão (preço, fotos, reviews)
→ Produto com CTR baixo e ROAS baixo = problema de relevância (errado para a keyword)

PRINCÍPIO 3 — MÚLTIPLOS TIPOS DE ADS MULTIPLICAM RESULTADO
→ Dados da Shopee: vendedores usando 2+ tipos de ads têm crescimento médio
  2,5x maior que quem usa apenas 1 tipo
→ Search Ads: captura quem já está buscando (intenção declarada)
→ Discovery Ads: captura quem não estava buscando mas pode comprar
  (expansão de audiência)
→ Não é "Search vs. Discovery" — é Search + Discovery em fases diferentes
→ Distribuição recomendada Feminnita:
  60% Search Ads (intenção direta) + 30% Discovery (alcance) + 10% Shop Ads

PRINCÍPIO 4 — KEYWORD RUIM É MAIS CARA QUE AUSÊNCIA DE KEYWORD
→ Keywords sem conversão consomem budget e ensinam o algoritmo que seu produto
  não converte — duplo prejuízo
→ Revisão de keywords: a cada 7 dias nas primeiras 4 semanas
→ Keyword com 10+ cliques e 0 conversão → pausar ou remover
→ CTR > 1,5% = sinal de relevância (o comprador considerou o produto)
```

## MÉTRICAS ATUAIS (2025)

```
ROAS (Return on Ad Spend = GMV gerado / gasto em ads)
Shopee usa janela de atribuição de 7 dias após o clique.

→ ROAS < 2x    → Pausar e auditar: problema no produto ou no targeting
→ ROAS 2x–4x  → Campanha viva mas ainda em ajuste — não escalar ainda
→ ROAS 4x–8x  → Zona saudável — manter budget, otimizar ROAS alvo gradualmente
→ ROAS > 8x   → Sub-investimento: o produto converte bem mas está com budget
                 apertado — aumentar orçamento em 30–50%
→ ROAS > 2x + budget do dia não sendo gasto → ROAS alvo alto demais
  → reduzir meta de ROAS para o algoritmo ampliar a entrega

CTR (Click-Through Rate)
→ CTR > 1,5% = compradores consideram o produto relevante
→ CTR < 1%    = problema na foto de capa ou no título do produto
→ Ação antes de aumentar bid: corrigir a apresentação do produto

CR — Conversion Rate (taxa de conversão)
→ Referência moda Shopee BR: 2–5%
→ CTR alto + CR baixo = o anúncio atrai, a ficha do produto não fecha a venda
  (revisar fotos, avaliações, preço, frete)

CPC (Custo por Clique) — referência pijamas/moda feminina
→ Referência atual: R$0,80–R$2,50
→ No GMV Max, o CPC é gerenciado pelo algoritmo — você define ROAS alvo,
  não bid direto
→ CPC fora da faixa em campanha manual/Search Ads: revisar competitividade
  da keyword
```

## ESTRUTURA DE CAMPANHA

```
HIERARQUIA:
Campanha (1 por objetivo) → Grupo de Anúncio (por tipo de produto) → Keywords/Produtos

NOMENCLATURA:
[Conta]_[Tipo]_[Fase]_[Produto]
Ex: FEM_Search_Crescimento_PijamaSuede | FEM_Discovery_Maturidade_BabydollVerao

GRUPOS:
→ Máximo 15 keywords por grupo (mais que isso = dados diluídos, diagnóstico difícil)
→ 1 tipo de produto por grupo (misturar categorias = difícil identificar o que converte)
→ Separar produto novo de produto maduro em grupos diferentes (objetivos diferentes)

BUDGET:
→ Mínimo R$30/dia por grupo para ter dados suficientes em 7 dias
→ R$50/dia = ponto ideal para aprendizado mais rápido do GMV Max
→ Aumentar budget sempre em etapas de 20–30% (nunca dobrar de uma vez)
```
