```

### 2. DESCRIÇÃO DE PRODUTO (estrutura padrão)

*Brian Dean: conteúdo superior ao que já existe. Neil Patel: keyword nos primeiros 100 caracteres.*

**Estrutura obrigatória — 4 parágrafos, ~300 palavras:**

```
PARÁGRAFO 1 — KEYWORD + PROPOSTA DE VALOR (keyword nos primeiros 100 caracteres)
→ Apresentar o produto com a keyword principal e o benefício imediato
→ Ex: "Pijama suede feminino atacado para revendedoras — kit com [X] peças
   em suede macio, ideal para revenda com alta margem de lucro."

PARÁGRAFO 2 — ESPECIFICAÇÕES TÉCNICAS
→ Tecido, composição, tamanhos disponíveis, quantidade do kit
→ Inclui variações de keyword naturalmente: "suede canelado", "pijama longo",
  "plus size atacado"
→ Responde a dúvida do lojista: o que exatamente estou comprando?

PARÁGRAFO 3 — BENEFÍCIO PARA REVENDA (argumento B2B)
→ Por que esse produto gira? Margem, prazo de entrega, sazonalidade
→ Inclui long tail: "pijama que vende no inverno", "produto com alto giro"
→ Responde a dúvida da revendedora: vou conseguir vender isso?

PARÁGRAFO 4 — CONDIÇÕES COMERCIAIS + CTA
→ Pedido mínimo, formas de pagamento, prazo de entrega
→ CTA claro: "Adicione ao carrinho e receba em [X] dias úteis"
→ Keyword de cauda longa: "fornecedor pijama suede com entrega rápida"
```

**Exemplo de descrição pronta (modelo):**

```
Pijama suede feminino atacado — kit com 6 peças disponível para revendedoras

O pijama suede é um dos produtos com maior giro no atacado de moda íntima —
com margem de revenda média de 80% e alta demanda no período de outono e inverno.

### 3. KEYWORD STRATEGY — LONG TAIL PARA ATACADO

*Neil Patel: keywords com intenção transacional têm CPC alto — são as mais valiosas.*
*Rand Fishkin: intenção de busca em 3 camadas — informacional, comercial, transacional.*

```
MAPA DE KEYWORDS POR PERFIL:

LOJISTA (intenção transacional alta):
→ "fornecedor pijama suede atacado"
→ "confecção pijama feminino preço de fábrica"
→ "atacado pijama adulto com nota fiscal"
→ "pijama suede atacado mínimo baixo"
→ "fabricante pijama feminino brasil"

REVENDEDORA (intenção comercial/transacional):
→ "pijama atacado para revendedora"
→ "kit pijama feminino para revender"
→ "pijama suede atacado para revenda whatsapp"
→ "fornecedor pijama com baixo pedido mínimo"
→ "pijama atacado margem de lucro"

AMIGAS / GRUPO (intenção transacional de preço):
→ "pijama preço de fábrica"
→ "comprar pijama direto da fábrica"
→ "pijama barato atacado"
→ "pijama feminino kit atacado 199 reais"
→ "pijama atacado sem CNPJ"

PRODUTO-ESPECÍFICAS (long tail de produto):
→ "pijama suede feminino atacado"
→ "pijama babydoll atacado"
→ "pijama infantil atacado revendedora"
→ "camisola atacado fornecedor"
→ "pijama plus size atacado"
→ "short doll suede atacado"
```

**Regra de uso:**
```
→ Cada página de produto = 1 keyword principal + 2–3 variações long tail
→ Nunca repetir a mesma keyword principal em 2 páginas diferentes
   (canibalização de keywords — páginas concorrem entre si no Google)
→ Página de categoria = keyword mais ampla
→ Página de produto = keyword mais específica
```

### 4. TOPICAL AUTHORITY — CONSTRUIR AUTORIDADE NO TEMA (Rand Fishkin)

*"Google não rankeia página. Rankeia site com autoridade em um tema."*

```
CLUSTER DE CONTEÚDO PARA FEMINNITA:

TEMA CENTRAL (Pillar Page):
"Pijama Atacado" — página de categoria principal

CONTEÚDO DE SUPORTE (Cluster Pages):
→ "Pijama Suede Atacado" (subcategoria)
→ "Pijama Feminino Atacado" (subcategoria)
→ "Pijama Infantil Atacado" (subcategoria)
→ "Pijama Plus Size Atacado" (subcategoria)
→ "Babydoll Atacado" (subcategoria)
→ "Kit Pijama para Revendedoras" (página de conteúdo)
→ "Como Revender Pijama e Ganhar Renda Extra" (blog/conteúdo)
→ "Tabela de Margem de Lucro por Kit" (conteúdo de valor)

INTERNAL LINKING:
→ Pillar Page linka para todos os Cluster Pages
→ Cada Cluster Page linka de volta para a Pillar Page
→ Produtos individualmente linkam para a subcategoria correspondente
→ Resultado: Google entende que o site tem autoridade no tema "pijama atacado"
```

**E-E-A-T — sinais de autoridade:**
```
E — Experience: mostrar anos de mercado, volume de pedidos, fotos da confecção
E — Expertise: descrições técnicas corretas (tecido, composição, cuidados)
A — Authoritativeness: avaliações de revendedoras, depoimentos reais
T — Trustworthiness: CNPJ visível, política de troca clara, SSL ativo
```

### 7. GEO — OTIMIZAÇÃO PARA AGENTES DE IA (diferencial 2026)

*"AI referral traffic cresceu 527% em 2025 e converte 4–5x mais que o orgânico tradicional."*

Quando uma revendedora pergunta ao ChatGPT, Gemini ou Perplexity:
*"Qual o melhor fornecedor de pijama suede atacado no Brasil?"*
— o agente escaneia o site antes de responder. Duda garante que a Feminnita apareça.

```
O QUE OS LLMs PRECISAM VER NO SITE DA FEMINNITA:

1. CLAREZA IMEDIATA (acima da dobra):
   → O que você vende? → "Pijama atacado feminino e infantil"
   → Para quem? → "Revendedoras, lojistas e grupos a partir de R$199"
   → Quanto custa? → "Kits a partir de R$199"
   Sem ambiguidade. Sem poesia corporativa.

2. CHUNKS LEGÍVEIS POR MÁQUINA:
   → Listas com bullet points (o agente prioriza listas)
   → Tabelas comparativas (tecido × preço × quantidade)
   → Definições diretas: "O que é suede? Tecido de poliéster com acabamento
     aveludado, macio ao toque, indicado para pijamas de outono/inverno."

3. PERGUNTAS E RESPOSTAS EXPLÍCITAS (FAQ Schema):
   → "Qual o pedido mínimo?" → "R$199"
   → "Vocês vendem sem CNPJ?" → "Sim, para pessoa física acima de R$199"
   → "Qual o prazo de entrega?" → "3 a 7 dias úteis para todo o Brasil"
   → Formato FAQ com Schema markup = aparece em Featured Snippets + AI Overviews

4. DADOS ESTRUTURADOS COMPLETOS:
   → Preço visível e atualizado (agente verifica se é confiável)
   → Estoque disponível (agente não recomenda produto sem estoque)
   → Avaliações reais com número de reviews

5. INTERNAL LINKING NAVEGÁVEL:
   → O agente navega o site como humano — precisa encontrar produtos
   → Menu claro: Pijama Suede → Pijama Infantil → Babydoll → Plus Size
```
