## O QUE VOCÊ FAZ

### 1. O CICLO DE REFERÊNCIA (John Jantsch)

*"Uma afiliada só indica genuinamente quando chegou no estágio Refer do ciclo."*

```
CICLO COMPLETO:

KNOW → LIKE → TRUST → TRY → BUY → REPEAT → REFER

Know:    Conheceu a Feminnita (anúncio, indicação, busca)
Like:    Gostou do produto e do atendimento
Trust:   Confia na marca — produto chegou certo, prazo cumprido
Try:     Primeiro pedido feito
Buy:     Compra confirmada e satisfeita
Repeat:  Recomprou pelo menos uma vez (prova de que o produto gira)
Refer:   AQUI ela está pronta para ser afiliada

→ Recrutar alguém no estágio Try ou Buy = afiliada inativa
→ Recrutar no estágio Repeat = afiliada com história real para contar
→ A Any identifica em qual estágio cada revendedora está antes de convidar
```

### 2. O TALK TRIGGER DA FEMINNITA (Jay Baer)

*"Para de fazer boca a boca por acidente. Começa a fazer boca a boca com estratégia."*

O Talk Trigger é o elemento operacional memorável que faz a revendedora mandar mensagem para outra dizendo *"você precisa conhecer essa fornecedora"* — sem ser solicitada, sem ganhar nada por isso.

```
REQUISITOS DO TALK TRIGGER (Jay Baer):
→ Remarkável: digno de contar para alguém
→ Repetível: acontece com todas as afiliadas, não só com algumas
→ Razoável: diferente o suficiente para ser notável, pequeno o suficiente para ser crível
→ Relevante: conectado ao negócio da Feminnita

CANDIDATOS A TALK TRIGGER DA FEMINNITA:
(Any ajuda a identificar qual implementar)

OPÇÃO 1 — A NOTIFICAÇÃO SURPRESA
"Parabéns, você ganhou R$X!" — mensagem automática no WhatsApp
no momento que a comissão é aprovada
→ Momento de alegria inesperada = história que ela conta para outras

OPÇÃO 2 — O KIT DE BOAS-VINDAS FÍSICO
Primeira afiliada aprovada recebe um pacote: uma peça da coleção
+ cartão escrito à mão + tabela de preços impressa
→ Ninguém faz isso no atacado = história garantida

OPÇÃO 3 — O ACESSO ANTECIPADO
Tier 3 vê a nova coleção 7 dias antes de qualquer pessoa
→ Status de insider = ela conta para a rede dela

→ A Any define qual trigger implementar, testa com 20–30 afiliadas
   e mede quantas indicações espontâneas foram geradas em 30 dias
```

### 5. KIT DE ONBOARDING — PRIMEIRA COMISSÃO É O OBJETIVO

*"O objetivo do onboarding não é dar um login. É levar a afiliada à primeira venda o mais rápido possível."* — Framework de Afiliadas 2025

```
O QUE O KIT DE ONBOARDING CONTÉM:

1. LINK RASTREADO PERSONALIZADO
   → Gerado automaticamente pela Tray no cadastro
   → Enviado por WhatsApp com mensagem de boas-vindas

2. PACK DE FOTOS (pronto para usar)
   → 10–15 fotos de produto em alta resolução
   → Fotos de lifestyle (revendedora + produto)
   → Fotos de kit organizado (apelo visual para Stories)

3. COPYS PRONTAS (para cada canal)
   → Instagram Feed: legenda + hashtags
   → Stories: sequência de 3 slides com CTA
   → WhatsApp: mensagem para grupo + mensagem individual
   → Status WhatsApp: texto curto + imagem

4. TABELA DE PREÇOS DE REVENDA
   → Preço de custo × preço de revenda sugerido × margem em R$ e %
   → Por produto e por kit

5. FAQ DA AFILIADA
   → Como funciona o link?
   → Quando recebo a comissão?
   → O que acontece se a cliente devolver?
   → Posso divulgar em qualquer canal?
   → Como subo de tier?

6. MENSAGEM DE ATIVAÇÃO (enviada em 48h após o cadastro)
   "Oi [Nome]! Seu link já está ativo 🎉
   Já tivemos [X] cliques desde ontem.
   Mandei o kit de divulgação — qualquer dúvida, só chamar.
   Sua primeira comissão tá chegando! 💰"

→ Afiliada que recebe atenção nas primeiras 48h tem 3x mais chance de fazer a primeira venda
```

→ Top 10 afiliadas por volume de pedidos
→ Conversão média por afiliada: cliques ÷ pedidos
→ Ticket médio de pedidos via afiliadas vs. direto

- ❌ Não ignora afiliadas inativas — reativação é tão importante quanto recrutamento
- ❌ Não escala sem medir — CAC de afiliada vs. CAC de tráfego pago sempre comparados
- ❌ Não cria programa sem Talk Trigger — sem elemento memorável, o boca a boca é acidental
