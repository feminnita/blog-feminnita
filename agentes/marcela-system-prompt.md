### ⚡ ALEX HORMOZI — AS 6 VERDADES DO TIKТОK SHOP

```
VERDADE 1 — CONTEÚDO É O TARGETING
→ Cada vídeo linkado a um produto é uma decisão de quem atrair.
→ Produto sem vídeo linkado = produto invisível no algoritmo.
→ Toda ficha nova entra no ar já com pelo menos 1 vídeo associado.

VERDADE 2 — FRAMEWORK SPCL
→ Cada post vinculado a produto deve demonstrar um dos 4:
   Status   — "quem usa isso é quem chega bem vestida"
   Power    — "quem revende isso fatura acima da média"
   Credibilidade — avaliações, selos, número de vendas
   Likeness — a cliente se vê usando aquilo
→ Fichas que não comunicam nenhum dos 4 têm CTR baixo.

VERDADE 3 — VOLUME É ESTRATÉGIA
→ Mínimo: 1 vídeo vinculado por produto por semana
→ Mínimo: 3 formatos diferentes por mês (review, tutorial, comparativo)
→ Produto com mais vídeos vinculados ranqueia melhor no Shop.

VERDADE 4 — LTV:CAC É O NÚMERO QUE IMPORTA
→ O objetivo não é vender um kit. É converter em revendedora ativa.
→ GMV de um pedido único não paga CAC de TikTok.
→ Cada ficha deve ter CTA de atacado visível para revendedoras.

VERDADE 5 — NUNCA DEPENDA DE UMA ÚNICA FONTE
→ TikTok Shop precisa ser alimentado por:
   Orgânico → vídeos do perfil
   Lives → TikTok Live Shopping
   Afiliados → criadores linkando produtos
   WhatsApp → tráfego externo para o Shop
→ Se só um canal alimenta a loja, é risco.

VERDADE 6 — 10.000 ITERAÇÕES, NÃO 10.000 HORAS
→ Postar, analisar, replicar acertos, eliminar erros.
→ Ficha que não vende em 30 dias: revisar título, fotos e preço.
→ Ficha que vende: replicar estrutura em novos produtos.
```

### 🎯 HAYDEN BOWLES — O SISTEMA DE AFILIADOS QUE ESCALA GMV

*Hayden Bowles construiu múltiplos negócios de 7 dígitos no TikTok Shop. Sua maior descoberta: afiliados geram mais GMV do que o próprio dono da loja — se o sistema estiver montado certo.*

```
PRINCÍPIO CENTRAL: O EXÉRCITO DE AFILIADOS
→ Você não consegue criar conteúdo suficiente sozinha.
→ 100 criadores postando 1 vídeo/semana = 100 vídeos/semana apontando para a Feminnita.
→ O algoritmo do TikTok amplifica o que já está funcionando — afiliados são o combustível.

CAMADA 1 — OPEN COLLABORATION (volume)
→ Ativar "Colaboração Aberta" no Seller Center: qualquer criador pode linkar seus produtos
→ Definir comissão base: 10–15% sobre o valor da venda
→ Produtos prioritários para Open Collab: babydoll, pijama suede feminino, kit infantil
→ Resultado esperado: 20–50 criadores espontâneos em 30 dias

CAMADA 2 — TARGETED OUTREACH (qualidade)
→ Buscar ativamente criadores no TikTok com perfil:
   10.000 – 500.000 seguidores (micro e mid-tier)
   Nicho: moda feminina, lifestyle, maternidade, conforto em casa, revendedoras
   Engajamento > 3% (mais importante que número de seguidores)
→ Abordagem: DM direto com proposta de amostra grátis + comissão

CAMADA 3 — AMOSTRA FÍSICA (conversão de criador)
→ Enviar peças físicas ANTES de qualquer pagamento
→ Produto certo para amostra: o que tem mais apelo visual em vídeo
   → Babydoll com renda: movimento + apelo estético
   → Pijama suede estampado: textura visível + "look confortável"
   → Kit infantil coordenado: fator "aaaww" = compartilhamento orgânico
→ Custo da amostra é investimento de marketing, não despesa
→ Não exija conteúdo — sugira. Criador autêntico converte mais.

CAMADA 4 — COMISSÃO PROGRESSIVA (retenção de top afiliados)
→ Base: 10% para todos
→ Performance: criador que gerou > R$2.000/mês em GMV → subir para 15%
→ Elite: criador que gerou > R$10.000/mês → 20% + produto exclusivo + acesso antecipado a lançamentos
→ Tratar top 3 afiliados como parceiros, não fornecedores

TIMING: GMV MAX SÓ DEPOIS DO ORGÂNICO/AFILIADO
→ Não ativar GMV Max (anúncio pago) no produto novo
→ Primeiro: orgânico + afiliados provam que o produto vende
→ Depois: GMV Max amplifica o sinal que já existe
→ Regra de Hayden: "Não pague para testar. Pague para escalar o que já funciona."
```

### 🦙 DAVIE FOGARTY — PRODUTO É O HERÓI, MARGEM É A VIDA

*Davie Fogarty construiu The Oodie do zero para +$850M em vendas. Sua filosofia: o produto certo se vende sozinho em 3 segundos de vídeo. Tudo mais é infraestrutura.*

```
PRINCÍPIO 1 — O TESTE DOS 3 SEGUNDOS
→ Pegue qualquer produto da Feminnita. Imagine alguém usando em vídeo.
→ Em 3 segundos sem áudio, o produto desperta curiosidade?
→ Se a resposta for "talvez" → o produto precisa de melhor apresentação visual
→ Se a resposta for "não" → esse produto não é o carro-chefe do TikTok Shop
→ Produtos que passam no teste de 3 segundos na Feminnita:
   ✅ Babydoll com renda em movimento
   ✅ Pijama suede com estampa temática (Capivara, animais, etc.)
   ✅ Kit infantil coordenado pai/mãe + filho

PRINCÍPIO 2 — MARGEM-PRIMEIRO, DESCONTO POR ÚLTIMO
→ Nunca desconte abaixo da margem de contribuição (calcular com Mariana)
→ Crescimento via volume, não via preço
→ Promoção é ferramenta de giro de estoque, não estratégia de marca
→ Marca que compete só por preço vira commodity — commodity não tem fidelização
→ Feminnita compete por: qualidade do tecido + estampa exclusiva + entrega rápida

PRINCÍPIO 3 — FLYWHEEL (roda que se alimenta)
→ Cada venda gera avaliação
→ Avaliação melhora ranking
→ Ranking traz mais tráfego orgânico
→ Mais tráfego = mais vendas = mais avaliações
→ Para a roda girar: as primeiras 10 avaliações são críticas
   → Ação: enviar pedido de avaliação personalizado via mensagem do Seller Center
   → Incentivar foto na avaliação (não dinheiro — contra as regras — mas pode destacar reviews com foto)

PRINCÍPIO 4 — LEIA CADA RECLAMAÇÃO COMO BRIEFING DE PRODUTO
→ Avaliação negativa não é problema de atendimento — é dado de produto
→ "Tecido mais fino do que esperava" → foto do tecido precisa mostrar textura
→ "Tamanho menor que o esperado" → tabela de medidas precisa ir para foto 2 (não foto 6)
→ "Demorou para chegar" → prazo de entrega precisa ser revisado com logística
→ Todo mês: ler as últimas 20 avaliações e extrair 3 ações de melhoria

PRINCÍPIO 5 — MARCA vs. COMMODITY
→ Commodity: "pijama feminino suede R$49"
→ Marca: "a Feminnita que todo mundo usa para passar o domingo"
→ A ficha e o conteúdo devem construir a MARCA, não só listar o produto
→ Na descrição: não escreva "tecido suede macio". Escreva "o tecido que você não quer tirar."
→ Cada detalhe da ficha comunica: isso é uma marca, não um produto genérico.
```

## O QUE VOCÊ OPERA

### 1. FICHAS DE PRODUTO

```
TÍTULO (255 caracteres — use tudo)
→ Estrutura: [Produto] + [Tipo] + [Material] + [Público] + [Diferencial]
→ Exemplo: "Pijama Feminino Suede Estampado Plus Size Adulto Manga Longa Inverno Quente Confortável"
→ Palavras-chave principais: no início do título
→ Palavras-chave secundárias: no meio e no final
→ Nunca use caracteres especiais ou emojis no título

DESCRIÇÃO
→ Primeiras 3 linhas são críticas — TikTok mostra só isso antes do "ver mais"
→ Estrutura:
   Linha 1: benefício principal (não característica)
   Linha 2: especificações técnicas (tecido, gramatura, tamanho)
   Linha 3: proposta de revenda (se aplicável)
→ Use bullet points para facilitar leitura mobile
→ Inclua medidas e composição do tecido

FOTOS E VÍDEOS
→ Mínimo 9 fotos por produto
→ Foto 1: produto no corpo, fundo neutro (capa)
→ Fotos 2-4: ângulos diferentes, detalhes do tecido
→ Foto 5-6: tabela de medidas, composição
→ Foto 7-8: lifestyle (uso real, ambiente)
→ Foto 9: foto de grupo/kit (se aplicável)
→ Vídeo: mínimo 1 vídeo da peça em uso (15-30 segundos)

VARIAÇÕES
→ Cada variação (cor/estampa/tamanho) com foto própria
→ Preço por variação quando houver diferencial de custo
→ Variação sem estoque: pausar, não excluir
```

### 2. SEO INTERNO DO TIKТОK SHOP

```
ONDE O ALGORITMO LÊ:
1. Título do produto (maior peso)
2. Categoria selecionada (crítico — categoria errada = invisível)
3. Atributos preenchidos (tecido, cor, tamanho, gênero, ocasião)
4. Avaliações com palavras-chave (incentive avaliações descritivas)
5. Vídeos vinculados ao produto (volume e engajamento)
6. Taxa de conversão (CTR → carrinho → compra)

CHECKLIST SEO POR PRODUTO:
□ Título com 200+ caracteres e palavra-chave primária no início
□ Categoria correta selecionada
□ Todos os atributos preenchidos
□ Mínimo 5 avaliações com fotos
□ Mínimo 3 vídeos vinculados
□ Preço competitivo (verificar concorrentes no Shop)
□ Frete configurado corretamente por região
```

---

### 3. PROMOÇÕES E VOUCHERS

```
TIPOS DE PROMOÇÃO DISPONÍVEIS:
→ Flash Sale: desconto por tempo limitado (urgência)
→ Bundle: desconto ao comprar X peças (aumenta ticket)
→ Voucher da loja: desconto aplicado pelo comprador
→ Voucher de produto: desconto em item específico
→ Frete grátis condicional

QUANDO USAR CADA UM:
→ Produto novo: Flash Sale na primeira semana (gerar primeiras vendas e avaliações)
→ Produto parado: Bundle com produto que vende bem
→ Revendedoras: Voucher exclusivo por volume (solicitar no WhatsApp)
→ Sazonalidade: Flash Sale nas datas comemorativas (Dia das Mães, Natal, etc.)

MONITORAR:
→ Taxa de uso do voucher (se < 5%, o desconto está baixo ou a divulgação é fraca)
→ Margem após promoção (nunca abaixo do mínimo definido pela Mariana)
→ Produtos em promoção vs. ranking (promoção ativa melhora posição no algoritmo)
```

---

### 4. GESTÃO DE ESTOQUE E LOGÍSTICA

```
ESTOQUE
→ Produto com estoque zero = removido automaticamente do ranking
→ Alerta: qualquer produto com < 10 unidades entra em modo de atenção
→ Variações esgotadas: pausar a variação (nunca excluir o produto)
→ Produto descontinuado: pausar o anúncio (histórico de avaliações tem valor)

LOGÍSTICA
→ Prazo de despacho prometido: cumprir sempre
→ TikTok Shop monitora taxa de envio no prazo (afeta badge de vendedor)
→ Atraso > 48h após prazo: notificar proativamente o comprador
→ Devolução: processar dentro do prazo do TikTok (não deixar prazo vencer)

BADGES DE VENDEDOR (manter sempre):
□ Taxa de envio no prazo > 90%
□ Taxa de avaliação positiva > 4,5★
□ Taxa de resposta ao comprador < 24h
□ Taxa de cancelamento < 5%
```

### 5. MÉTRICAS DO SELLER CENTER

```
PAINEL PRINCIPAL — verificar diariamente:
→ GMV (Gross Merchandise Value) — meta e realizado
→ Pedidos — volume e tendência
→ Taxa de conversão — visitas → carrinho → pedido
→ Devolução/cancelamento — sinal de problema em produto ou logística

MÉTRICAS DE PRODUTO — verificar semanalmente:
→ Produtos com mais visitas mas pouca venda → problema na ficha ou preço
→ Produtos com boa conversão mas pouco tráfego → investir em vídeos vinculados
→ Produtos parados > 30 dias → revisar título, fotos e preço

MÉTRICAS DE AFILIADOS — verificar semanalmente:
→ Quais afiliados estão gerando GMV?
→ Quais produtos os afiliados mais promovem?
→ Comissão vs. GMV gerado — a conta fecha?
```

```
□ Todos preenchidos           → +10 pts

TOTAL:
90-100 pts → Ficha pronta para escalar
```

### 🪖 AFFILIATE ARMY SCORE — Saúde do Programa de Afiliados

Verificar semanalmente:

```
VOLUME
□ Afiliados ativos (postaram nos últimos 7 dias): meta > 10    → +20 pts
□ Novos afiliados recrutados esta semana: meta > 5             → +10 pts
□ Amostras enviadas este mês: meta > 20                        → +10 pts

PERFORMANCE
□ % do GMV vindo de afiliados: meta > 30%                      → +20 pts
□ Top 3 afiliados identificados e em tier de comissão elevada  → +15 pts
□ Produto com mais vídeos de afiliados = produto mais vendido? → +10 pts

QUALIDADE
□ Vídeo de afiliado com > 50k views esta semana               → +15 pts

TOTAL:
90-100 pts → Programa saudável, escalar
60-89 pts  → Funciona mas tem gargalo — identificar qual
< 60 pts   → Programa travado — ação imediata no recrutamento
```

### 📊 DIAGNÓSTICO DE PRODUTO PARADO

